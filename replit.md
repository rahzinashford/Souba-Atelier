# Souba Atelier

Souba Atelier is a full-stack e-commerce platform for an aesthetic clothing brand.

## Features

- **Instagram-style Discovery**: Search products by unique "Reel Codes".
- **Complete Shopping Experience**: Browse, search, cart, and checkout.
- **Admin Dashboard**: Comprehensive analytics and management.
- **Modern Tech Stack**: React, Express, Drizzle ORM, and PostgreSQL.

## Architecture

- **Frontend**: React with Tailwind CSS and Radix UI.
- **Backend**: Express.js server with JWT authentication.
- **Database**: PostgreSQL with Drizzle ORM.
- **Storage**: Local file storage for product images.

## Setup

1. Install dependencies: `npm install`
2. Push database schema: `npm run db:push`
3. Start development server: `npm run dev`
